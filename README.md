# keycloak-playground
A sandbox environment to explore Keycloak.

## Quick Start
Before proceeding, verify that port 8080 is not in use on your system.
1. Launch keycloak
    ```bash
    make all
    ```

2. To clean up, run the following command
    ```bash
    make clean
    ```
